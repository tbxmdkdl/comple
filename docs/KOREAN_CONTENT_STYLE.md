# KOREAN_CONTENT_STYLE.md

## 한국어 톤

사용자-facing 게임 콘텐츠는 자연스럽고 간결한 한국어로 작성한다. 말투는 전문적이지만 딱딱하지 않게 유지한다.

- 실무자가 바로 이해할 수 있는 표현을 쓴다.
- 과장된 위협, 훈계, 법률적 단정을 피한다.
- 짧은 문장으로 선택의 이유와 결과를 보여준다.
- 플레이어를 비난하지 않고 다음 판단을 돕는다.
- Code identifiers, filenames, commands, package names, APIs는 English를 유지한다.

## 카드 이름 규칙

- 카드 이름은 짧고 행동이 떠오르게 작성한다.
- 가능하면 2~6어절 이내로 유지한다.
- 추상적인 교훈보다 구체적인 실무 행동을 우선한다.
- 같은 행동 계열은 용어를 일관되게 쓴다.
- 카드 이름은 특정 상업용 카드 게임의 card name, combat term, signature wording을 연상시키면 안 된다.

좋은 예:

- 사실관계 확인
- 정책 확인
- 요청 내용 기록
- 안전한 채널 제안
- 이해상충 공개
- 승인 요청
- 보복 우려 공유

피할 예:

- 완벽한 윤리적 선택
- 법적으로 무조건 안전한 조치
- 정답 카드
- 처벌 회피
- 공격
- 방어
- 적 의도 파악

## 시나리오 작성 규칙

- 실제 업무 요청처럼 시작한다.
- 누가, 무엇을, 왜 요청하는지 빠르게 이해할 수 있어야 한다.
- 일정, 관계, 성과, 상급자, 고객, 비용 같은 압박을 포함한다.
- 숨은 리스크 신호를 하나 이상 넣는다.
- 플레이어가 여러 대응 전략을 선택할 수 있게 만든다.
- 특정 법률 조항이나 관할권에 의존하지 않는다.
- 한 시나리오 안에서 너무 많은 이슈를 한꺼번에 넣지 않는다.
- 전투, 적, 공격, 체력 같은 combat framing을 사용하지 않는다.

## 컴플라이언스 피드백 작성 규칙

- 피드백은 1~2문장으로 충분하다.
- 행동의 효과와 리스크 관리 이유를 설명한다.
- "항상", "절대", "불법" 같은 단정 표현을 신중히 사용한다.
- 법률 자문처럼 들리는 결론을 피한다.
- 다음 행동으로 이어지는 실무 힌트를 제공한다.
- 플레이어의 실패도 학습 가능한 정보로 다룬다.
- "정답/오답"보다 tradeoff와 다음 판단 포인트를 설명한다.

## 일관되게 사용할 용어

- risk: 리스크
- evidence: 증거
- trust: 신뢰
- pressure: 압박
- attention: 주의력
- time: 시간
- scenario: 시나리오
- card: 카드
- deck: 덱
- hand: 손패
- discard: 버림
- event: 이벤트
- reward: 보상
- passive: 패시브
- escalation: 에스컬레이션
- learning feedback: 학습 피드백
- final report: 최종 리포트
- compliance: 컴플라이언스

게임 내 표시명은 필요에 따라 한국어를 우선하되, 개발용 type name이나 function name은 English를 유지한다.

## 피해야 할 용어와 표현

- 법률 자문처럼 들리는 표현
- 특정 국가나 관할권의 법적 결론
- 플레이어를 꾸짖는 표현
- "정답", "오답"만 표시하는 피드백
- 공포를 조장하는 과장된 결과
- 실제 기업, 실제 사건, 실제 인물처럼 보이는 구체명
- 특정 상업용 카드 게임을 연상시키는 고유 용어
- enemy, attack, block, damage, combat 같은 전투 중심 표현
- "무조건 안전", "완전히 합법", "즉시 처벌"처럼 단정적인 표현

## 좋은 예시

- 요청 내용을 먼저 기록하면 이후 검토에서 사실관계를 설명하기 쉬워집니다.
- 정책을 확인해 두면 빠른 처리보다 안전한 처리 기준을 세울 수 있습니다.
- 압박이 높을수록 혼자 판단하기보다 근거를 모아 상의하는 편이 안전합니다.
- 안전한 채널을 제안하면 업무 흐름을 유지하면서 개인정보 노출 리스크를 줄일 수 있습니다.
- 이해상충 가능성을 일찍 공개하면 신뢰 손상을 줄이고 대안을 찾기 쉽습니다.

## 나쁜 예시

- 정답입니다!
- 틀렸습니다. 이런 행동은 하면 안 됩니다.
- 이 조치는 무조건 불법입니다.
- 당신은 컴플라이언스를 위반했습니다.
- 법적으로 완전히 안전한 선택입니다.
- 회사가 큰 처벌을 받을 수 있으니 즉시 겁내야 합니다.
- 적의 공격을 방어했습니다.

## 법률 자문처럼 들리지 않기

모든 콘텐츠는 일반적인 교육용 예시다. 특정 사건에 대한 법률 판단, 조사 결론, 징계 판단, 관할권별 법적 의무처럼 작성하지 않는다.

표현은 "리스크를 줄일 수 있습니다", "검토에 도움이 됩니다", "확인이 필요합니다", "상황에 따라 추가 확인이 필요할 수 있습니다"처럼 실무적이고 조건부인 문장을 우선한다.

## SME 검토 필요

실제 배포 전에는 모든 시나리오, 카드, 피드백, 최종 리포트 문구를 legal/compliance subject-matter experts가 검토해야 한다.

MVP 콘텐츠는 학습 방향을 잡기 위한 초안이며, 실제 정책이나 법률 요구사항을 대체하지 않는다.
